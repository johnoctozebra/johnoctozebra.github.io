Phishing Summary (blog)

Bank of America .docx Trojan

User is sent a phishing email with someone pretending to be from Bank of America with a hyperlink to what appears to be a Bank of America card statement in a word document. This takes the user to:

http://www.lnne.com.br/api/get.php?id=ZWRnYXIuc2lsdmFAdW5pdGVkLmNvbQ==

which automatically downloads the .docx. This only works on Windows system. The file seems to self-delete if not on a Windows system. If the user is on a Mac or Linux the page will give a 404 error and will not download.

Email:

The document contains an image (in a crude way) that something went wrong and instructs the user to enable editing, then enable the macro. On my VM, nothing happens when running after enabling the macro. VirusTotal gives 7/64 as a trojan and gives the download URL 2/64 malicious.

Word Document:

To view the macro code,
•open the Word document (preferably in a VM)
•Click Enable Editing
•Hold shift (disables auto open), then click Enable Content
•Then press alt + f11 (on mac, also hold the fn key)

Macro code contains repeated foul languages in what appears to be song lyrics.

becharaissa.com

This site pretends to be Royal Bank of Scotland and phishes for the user's Debit, Pin, and personal information such as date of birth on a WordPress site. Javascript softly verfies if fields are numbers or emails end in a valid TLD from a short list. After phishing is complete, the last page takes you to the legitimate Royal Bank of Scotland website.
